### 介绍
基于.net core 3.1 的微服务开发框架,使用简化的DDD+CQRS设计

### framework 微服务框架解决方案
* **Qf.Core**
    核心类库,部分代码参考自[abp](https://github.com/abpframework/abp)项目,用来实现框架的基础功能
* **Qf.Core.EFCore**
    基于EntityFrameworkCore的仓储基类
* **Qf.Core.Web**
